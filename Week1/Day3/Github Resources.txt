Basic Terminal Commands:
Display current directory: pwd
Create durectory: mkdir directoryname
Remove an empty directory: rmdir directoryname
Change to directory: cd directoryname or path/to/directory/
go back: cd ..
Lists the contents of a directory: ls
Create File: touch filename.extension
Open file with vsCode: code filename
Clear Terminal: clear





Basic Git Commands:
git init (initialize repository "1 time")
git remote add origin url (connect to your repository  "1 time")
git remote -v (Check if there is connection with the url)
git status (check the status of your tracking files in your repository)
git add . (track the files)
git status (check the status of your tracking files in your repository)
git commit -m "message" (commit your update)
git status (check the status of your tracking files in your repository)
git push origin master (send to github repository)
